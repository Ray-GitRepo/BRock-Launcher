## What is this

I make Minecraft texture packs but testing them meant exporting to my phone every time — too slow. I run Linux, so I looked for a way to play Bedrock natively. I found [NIX Launcher (mcpelauncher)](https://github.com/minecraft-linux/mcpelauncher-manifest) — it's great and still the core of how this works — but I wanted more control. Custom GUI, mod support, account management, the works.

BRock started as a wrapper around NIX Launcher. Now it's growing into something bigger.

---

## Goals

The end goal is a full Bedrock client for Linux with the feel of Lunar Client — polished UI, mod support, and account management — built on top of `mcpelauncher-client` by the NIX team.

The custom client layer is called **Gamma** — currently in development.

---

## Features

- Import and manage Bedrock versions from `.apk`
- Resource pack, behaviour pack, and world management
- Built-in server host and control panel
- 8 built-in themes with live sidebar art
- Custom backgrounds and CSS theme support
- Animated splash screen
- Keyboard shortcuts with full rebind support
- Storage search
- Game log viewer with filtering
- Profile with custom picture and playtime tracking

---

## Requirements

- Linux (x86_64)
- [mcpelauncher-client](https://github.com/minecraft-linux/mcpelauncher-manifest) installed via Flatpak
- Python 3
- GTK 3
